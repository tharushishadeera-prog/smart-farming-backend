# Smart Farming Management System - Backend

## Project Description

The backend of the Smart Farming Management System is developed using Node.js, Express.js, TypeScript, and MongoDB. It provides secure REST APIs for user authentication, crop management, harvest tracking, financial management, weather information, reporting, and AI-powered farming assistance using Google Gemini AI.

## Technologies Used

* Node.js
* Express.js
* TypeScript
* MongoDB
* Mongoose
* JWT Authentication
* bcryptjs
* Google Gemini API

## Features

* JWT-based Authentication and Authorization
* Crop Management APIs
* Harvest Management APIs
* Expense Management APIs
* Dashboard Analytics APIs
* Weather Information APIs
* AI Farming Assistant APIs
* Report Generation APIs
* User and Admin Management

## Environment Variables

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_ai_api_key
PORT=5000
```

## Setup Instructions

```bash
# Clone the repository
git clone <backend_repository_url>

# Install dependencies
npm install

# Run development server
npm run dev
```
